== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate "Ni Show Product Name In Orders Tab Of My Account Page For WooCommerce" plugin from admin panel.
3. Go to my account page in order tab and find column "Product | Quantity".
